# 🎯 Marketing Copywriter for 55+ Audience

Проект Claude Code для создания рекламного контента, обходящего рекламную слепоту.

## Быстрый старт

1. Скопируй содержимое этой папки в корень своего проекта
2. Запусти `claude` в терминале
3. Используй команды ниже

## Структура проекта

```
marketing-copywriter-project/
├── CLAUDE.md                    # Основной промпт (загружается автоматически)
├── .claude/
```

## Доступные команды

### `/analyze [текст или ссылка]`
Глубокий анализ рекламного поста: структура, триггеры, почему работает.

```
/analyze "Мне 67 лет, и я хочу рассказать как избавилась от боли в коленях..."
```

### `/generate [формат] [тема]`
Создание нового поста в указанном формате.

Форматы:
- `ugc` — личная история от обычного человека
- `medical` — от лица врача/эксперта
- `hybrid` — личная история + авторитет врача
- `investigation` — журналистский формат

```
/generate ugc суставы
/generate medical давление
/generate hybrid похудение
```

### `/warmup-series [тема] [количество касаний]`
Создание серии прогревающих постов для холодной аудитории.

```
/warmup-series витамины для зрения 5
```

### `/script [формат] [хронометраж] [тема]`
Создание видео-сценария с раскадровкой.

```
/script ugc 60s суставы
/script medical 90s память
```

### `/compare-formats [ниша]`
Сравнение эффективности форматов для конкретной ниши.

```
/compare-formats БАДы для суставов
```

## Как работать с примерами

1. Помести успешные посты в папку `examples/`
2. Назови файлы понятно: `ugc-суставы-успех.txt`, `medical-давление.md`
3. Claude автоматически проанализирует их и выявит паттерны
4. При генерации нового контента будет использовать паттерны, а не копировать

## Форматы рекламы

### UGC (User Generated Content)
Выглядит как пост обычного человека. Максимальная нативность.
- ✅ Высокая виральность
- ✅ Низкое сопротивление
- ⚠️ Средняя конверсия

### Медицинский/научный
От лица врача или эксперта. Максимальный авторитет.
- ✅ Высокое доверие
- ✅ Хорошая конверсия
- ⚠️ Юридические риски
- ⚠️ Низкая виральность

### Гибридный
Личная история + подтверждение от врача.
- ✅ Баланс доверия и виральности
- ✅ Хорошая конверсия
- ✅ Умеренные риски

## Исследование конкурентов

Чтобы Claude провёл исследование рекламы конкурентов:

```
Проведи исследование рекламы [продукт] для аудитории 55+.
Найди примеры в форматах UGC и медицинском.
Определи, какой формат работает лучше.
```

## Этические ограничения

Claude автоматически:
- Не обещает медицинских результатов
- Не имитирует реальных врачей
- Добавляет дисклеймеры где нужно
- Не манипулирует страхом смерти

## Советы по использованию

1. **Всегда давай примеры** — чем больше примеров, тем точнее паттерны
2. **Уточняй аудиторию** — "женщины 60+ с проблемами суставов" лучше, чем "пенсионеры"
3. **Указывай платформу** — ВК и Одноклассники требуют разного тона
4. **Проси варианты** — "дай 3 варианта хука" для тестирования

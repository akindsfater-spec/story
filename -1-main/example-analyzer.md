# Правила анализа примеров

## Принцип: Извлекай паттерны, не копируй

При получении примера успешного поста НИКОГДА не используй:
- Те же самые фразы дословно
- Идентичную структуру предложений
- Прямые заимствования историй

ВСЕГДА извлекай:
- Почему этот хук работает (механика, не форма)
- Какую эмоцию вызывает каждый блок
- Темп раскрытия информации
- Соотношение истории и продажи
- Тип социального доказательства

---

## Обязательный анализ

Для каждого примера создай:

### 1. Структурная карта
```
[ХOOK] → [БОЛЬ] → [ПОИСК] → [ОТКРЫТИЕ] → [СКЕПСИС] → [ПРОБА] → [РЕЗУЛЬТАТ] → [CTA]
  ↓        ↓         ↓           ↓            ↓          ↓          ↓          ↓
[какой]  [какая]   [как]     [через кого]  [какой]    [как]     [какой]    [какой]
```

### 2. Эмоциональный график
```
Начало:     [страх|боль|любопытство|узнавание]
Середина:   [надежда|сомнение|интерес]
Кульминация:[облегчение|радость|уверенность]
Конец:      [доверие|желание|срочность]
```

### 3. Факторы успеха
- Главный фактор: [что сделало пост успешным]
- Вторичные: [поддерживающие элементы]
- Рискованные: [что могло не сработать]

### 4. Переносимые элементы
- ✅ Можно адаптировать: [что]
- ⚠️ С осторожностью: [что]  
- ❌ Не переносить: [слишком специфичное]

---

## Запрещённые действия

1. **Не выдавай анализ за готовый пост**
   - Анализ — отдельно
   - Генерация нового — отдельно

2. **Не смешивай примеры**
   - Анализируй каждый изолированно
   - Затем сравнивай

3. **Не игнорируй контекст**
   - Где был опубликован
   - Какая аудитория видела
   - Какой продукт продвигался

---

## Формат вывода анализа

```markdown
## 📊 Анализ примера: [краткое описание]

### Тип: [UGC | Медицинский | Гибрид | Журналистский]

### Структура
[структурная карта]

### Что работает
1. [элемент] — [почему работает]
2. [элемент] — [почему работает]

### Ключевой инсайт
> [одно предложение — главный урок из этого примера]

### Как применить
- Для [ситуация 1]: [как адаптировать]
- Для [ситуация 2]: [как адаптировать]

### ⚠️ Что НЕ копировать
- [элемент] — [почему не сработает в другом контексте]
```

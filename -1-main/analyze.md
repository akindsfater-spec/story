# Анализ рекламного примера

Проведи глубокий паттерн-анализ предоставленного рекламного поста.

## Вход
$ARGUMENTS

Если аргумент — ссылка, попробуй получить контент.
Если текст — анализируй напрямую.

---

## Алгоритм анализа

### Шаг 1: Классификация
- Определи формат: UGC / Медицинский / Гибрид / Журналистский / Другое
- Определи этап воронки: холодный / тёплый / горячий
- Определи целевую аудиторию внутри 55+

### Шаг 2: Структурный разбор
Разбей на блоки:
```
[ХУК] → [КОНТЕКСТ] → [БОЛЬ] → [ПЕРЕЛОМНЫЙ МОМЕНТ] → [РЕШЕНИЕ] → [ДОКАЗАТЕЛЬСТВО] → [CTA]
```

Для каждого блока:
- Сколько слов/предложений
- Какую функцию выполняет
- Какую эмоцию вызывает

### Шаг 3: Анализ триггеров
Выдели использованные психологические триггеры:
- [ ] Социальное доказательство
- [ ] Авторитет
- [ ] Дефицит/срочность
- [ ] Страх потери
- [ ] Надежда
- [ ] Принадлежность к группе
- [ ] Взаимность
- [ ] Любопытство
- [ ] Конкретика vs абстракция

### Шаг 4: Механика доверия
Как снимается скептицизм:
- Какие возражения предвосхищены
- Какие доказательства приведены
- Как подаётся "ненавязчивость"

### Шаг 5: Антирекламные элементы
Что делает пост "нерекламным":
- Стилистические приёмы
- Структурные решения
- Визуальные элементы (если описаны)

## Формат вывода

```markdown
# 📊 Паттерн-анализ

## Метаданные
- **Формат:** [формат]
- **Целевая аудитория:** [кто]
- **Этап воронки:** [какой]
- **Продукт/ниша:** [что продвигается]

## Структура (с хронометражем)
| Блок | Слова | Функция | Эмоция |
|------|-------|---------|--------|
| Хук  | N     | [функция] | [эмоция] |
| ...  | ...   | ...     | ...    |

## Эмоциональная дуга
[график: low → high, какие эмоции в какой момент]

## Триггеры доверия
1. [триггер] — [как реализован]
2. [триггер] — [как реализован]

## Ключевой инсайт
> [одно предложение — главный урок]

## Как применить этот паттерн
- Для [ситуация]: [адаптация]

## ⚠️ Что НЕ копировать
- [элемент] — [причина]
```

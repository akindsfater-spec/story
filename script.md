# Создание видео-сценария

Создай сценарий для рекламного видео в указанном формате.

## Параметры
$ARGUMENTS

Формат: `[формат] [хронометраж] [тема]`
Примеры:
- `ugc 60s суставы`
- `medical 90s давление`
- `hybrid 45s похудение`

Если хронометраж не указан, используй 60 секунд.

---

## Форматы видео

### UGC-видео (селфи-стиль)
**Визуал:** 
- Съёмка на телефон, "живые" кадры
- Домашняя обстановка
- Естественное освещение
- Возможны "ошибки" (отвлечение, поправка волос)

**Структура (60 сек):**
```
0-5 сек:   Хук-зацепка (боль или интрига)
5-15 сек:  Представление + "было"
15-30 сек: Переломный момент
30-45 сек: Результат + доказательства
45-55 сек: Рекомендация
55-60 сек: CTA
```

### Медицинский формат
**Визуал:**
- Кабинет/белый фон
- Белый халат (если врач)
- Графики/схемы на экране
- Спокойный, уверенный темп

**Структура (60 сек):**
```
0-5 сек:   Позиционирование эксперта
5-15 сек:  Проблема (что вижу у пациентов)
15-35 сек: Объяснение механизма
35-50 сек: Решение + результаты
50-60 сек: Рекомендация + CTA
```

### Гибридный формат
**Визуал:**
- Чередование: личная история + вставки эксперта
- Или: цитата врача на экране во время рассказа

## Шаблон сценария

```markdown
# Сценарий: [название]

**Формат:** [формат]
**Хронометраж:** [время]
**Целевая аудитория:** [кто]
**Платформа:** [где размещать]

---

## Раскадровка

### [00:00 - 00:05] Хук
**Визуал:** [что на экране]
**Текст на экране:** [если есть]
**Голос:**
> [реплика]

**Заметки для съёмки:** [как снимать]

---

### [00:05 - 00:15] [название блока]
**Визуал:** [описание]
**Голос:**
> [реплика]

---

[и так далее по блокам]

---

- [рекомендации по записи голоса]

**Монтаж:**
- [рекомендации по склейке]

## Варианты концовок

**Вариант A (мягкий):**
> [текст]

**Вариант B (с urgency):**
> [текст]

---

## Чек-лист перед съёмкой
- [ ] Естественное освещение
- [ ] Чистый бытовой фон
- [ ] Телефон на уровне глаз
- [ ] Тестовая запись звука
- [ ] Актёр понимает эмоции каждого блока
```

---

## Важно для UGC

1. **Не читать по бумажке** — импровизация по смыслу
2. **Делать паузы и "ээ"** — реалистичность
3. **Смотреть в камеру как на друга**
4. **Эмоции важнее текста**
5. **Несколько дублей разной подачи**

---

## Важно для медицинского

1. **Спокойный темп** — без спешки
2. **Жесты объяснения** — как на приёме
3. **Профессионализм** — но не отстранённость
4. **Простые аналогии** — сложное простым языком
